------------------------------------------------Multiple Regression--------------------------------------------------

In this project we have students dataset with four variables score, intelligence quotient(iq), hours of study and gender. We will try to analyse if the score is affected by iq and hours of study. Multiple regression is conducted between 1 dependent contonuous variable and 2 or more independent continuous variables.

variable types in our dataset score = Dependent(continuous) 1q = Independent (continuous) hours = Independent (continuous)

The p values for coefficicnts are also less than 0.05 hence, our coefficients are significant and not equal to 0 so both independent variable give a satisfactory explanation to the score.

Assumptions to be met before computing multiple regression are as follows:

  1. Relationship must be approximately linear
  2. Variables do not represent outliers
  3. There is independence of error
  4. There is no important multicollinearity
  5. There is homoskedasticity
  6. the residuals are normally distributed

#1. Relationship must be approximately linear
library('ggplot2')
#Trend line
#create model for linear model
model <- lm(score~iq,data=students)

#create min and max iq for predicting values
min <- min(students$iq)
max <- max(students$iq)

#create vector of two values
iq <- c(min,max)

#predict the values for the given iq
fit <- predict(model,data.frame(iq))

#create dataframe based on min and max values for iq and predicted values for score
endpoints <- data.frame(iq,fit)

#construct scatter plot with trend line
ggplot()+ geom_point(data=students,aes(x=iq,y=score),color='blue') +
geom_line(data=endpoints,aes(x=iq,y=fit),color='red')+
ggtitle("Scatter Plot")

Assumption 1 is met and we can see there is a linear relationship between the two variables.
